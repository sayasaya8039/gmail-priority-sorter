# Gmail Priority Sorter v1.0.0

AIがGmailの受信トレイを分析し、重要なメールを上位に並べ替え、緊急度スコアを自動付与するChrome拡張機能です。

## インストール方法

1. `chrome://extensions/` を開く
2. 「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択

## 主な機能

- 自動優先度分類（緊急/高/中/低）
- 緊急度スコア表示（0-100）
- カテゴリ分類（緊急、会議、要対応など）
- 自動ソート機能
- VIP/無視リスト管理
- ダークモード対応

## 使い方

1. Gmailを開く (https://mail.google.com)
2. 拡張機能が自動的にメールを分析
3. ポップアップから設定を調整可能

## プライバシー

すべての処理はローカルで行われ、メールデータがサーバーに送信されることはありません。
